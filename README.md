## 🏗️ Architecture

- **Frontend**: React + TypeScript + Vite (Vercel)
- **Backend**: FastAPI + Python 3.11 (Docker)
- **Database**: Supabase PostgreSQL with pgvector
- **AI Services**: OpenAI, Anthropic, Together AI, Replicate
- **Vector Search**: OpenAI embeddings (1536D), CLIP (512D)

## 🚀 Key Features

- **PDF Processing**: 14-stage AI pipeline with checkpoint recovery
- **Products + Metadata**: Inseparable extraction with JSONB storage
- **Document Entities**: Certificates, logos, specifications as separate knowledge base
- **Multi-Vector Search**: 6 specialized embeddings for comprehensive search
- **Agentic Queries**: Factory/group filtering for intelligent queries
- **Material Recognition**: Llama 4 Scout 17B Vision (69.4% MMMU, #1 OCR)
